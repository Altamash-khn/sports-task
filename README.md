# Sports Assignment

This is a **frontend assignment** built using **Vanilla HTML5** and **CSS3**.  
It showcases a **sports-themed web layout** designed for **1440px screen resolution**, demonstrating responsive layout handling and clean UI design principles.


## Live Demo
Click <a href="https://sports-assignment.vercel.app/">HERE</a> to see a live demo.


## Features

- Static sports webpage layout for 1440px screen width  
- Built entirely using **HTML5** and **CSS3** (no frameworks)  
- Organized folder structure with separate files for images and styles  
- Clean, minimalistic design suitable for visual presentation or assignment work

## Tech Stack

- **HTML5** — Semantic structure and layout  
- **CSS3** — Styling and page design
